## 引领未来 (LEADING THE WAY)

## 目的

本仓库旨在提供一份全面的资源清单，以造福 IIUM 学生。这包括学术资源、学生生活信息以及其他有用的链接。

## 如何使用

要使用本仓库，只需浏览提供的文件和文件夹。每个文件夹包含不同类别的资源。点击提供的链接即可访问相应资源。

## 贡献

我们要欢迎所有人的贡献。如果您有资源想要添加或有改进建议，请遵循以下的贡献指南。

### 贡献指南

1. Fork 本仓库。
2. 为您的更改创建一个新分支（branch）。
3. 在您的分支中进行更改。
4. 提交包含您更改的拉取请求（pull request）。

我们感谢您的贡献，并期待在您的帮助下看到本仓库不断壮大！

## 文件夹结构

- Academic Resources: 包含与学术相关的资源，如教科书、在线课程和学习指南。
- Student Life: 关于 IIUM 学生生活的信息，包括社团、活动和校园设施。
- Campus Facilities: 关于校园可用设施的详细信息，如图书馆、自习室和体育设施。
- Other Resources: 任何不属于上述类别的其他资源。

每个文件夹都包含一个 README 文件，其中包含有关该文件夹中资源的更多信息。

## IIUM 官方网站和门户

本部分提供了一系列官方网站和门户，为 IIUM 社区提供各种服务。

- [马来西亚国际伊斯兰大学 (IIUM) - 官方网站](https://www.iium.edu.my/): 包含关于大学、愿景、使命、历史、管理、学院、中心、研究所、校区与其它的常规信息。
- [i-Ma'luum - 学生服务](https://imaluum.iium.edu.my/): 一个 _基于网络_
  的系统，为学生提供**重要的在线服务**，如课程注册、考试成绩、财务报表、学术成绩单等。
- [i-Ta'leem - 学习管理系统](https://italeem.iium.edu.my/): 支持在线教学活动，如课程材料、作业、测验、论坛等。
- [i-Viva - 研究生 Viva 流程](https://iviva.iium.edu.my/): 管理研究生的 Viva 流程，如提名导师、提交研究计划、安排 Viva 会议等。

## IIUM 校友网络和职业机会

## IIUM 研究和出版物
